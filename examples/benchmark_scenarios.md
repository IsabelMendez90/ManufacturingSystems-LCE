# Benchmark scenarios

Use the same objective, industry, role, selected stages, and current I5S levels across the three full-framework runs. Vary only the manufacturing-system typology.

## Shared scenario

- Objective: Design and ramp a flexible small-batch manufacturing system for a precision micromachine/component product, including supply-chain configuration, production planning, quality control, sensing capability, smart manufacturing support, lifecycle sustainability, and safe operation.
- Industry: Precision component manufacturing
- Role: Manufacturing systems engineer
- Current I5S maturity: Social 1, Sustainable 1, Sensing 1, Smart 1, Safe 1
- LCE stages: Ideation, Basic Development, Advanced Development, Launching, End-of-Life
- Uploaded documents: none
